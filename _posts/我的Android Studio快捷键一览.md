## 我的Android Studio快捷键一览

|     功能      |      快捷键       |
| :---------: | :------------: |
|    智能提示     |    Ctrl + ,    |
|    自动修正     |    Alt + 回车    |
|     删除行     |    Ctrl + Y    |
|    重写方法     |    Ctrl + O    |
|    格式化代码    | Ctrl + Alt + L |
|   清楚无效引用包   | Alt + Ctrl + O |
|   查找（替换）    |  Ctrl + F (R)  |
|    封装成方法    | Ctrl + Alt + M |
| 常用插件和生成相关代码 | Alt  + Insert  |
